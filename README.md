# 🏠 Real Estate Data Analysis (SQL + Power BI)

## 📊 Overview
End-to-end analysis of US real estate data using SQL and Power BI to uncover pricing trends, market size, and property distribution.

## 🔧 Process
1. Data Cleaning using SQL
2. Data Transformation (feature engineering)
3. Exploratory Data Analysis (SQL)
4. Dashboard creation in Power BI

## 🧹 Data Cleaning (SQL)
- Removed duplicates
- Handled null values
- Fixed inconsistent city names (e.g., Ny → New York)
- Converted data types
- Created new features:
  - Year
  - House size (m²)
  - Lot size (hectares)

## 📈 Key Insights
- New York has highest market size (~21bn)
- New Jersey has highest property count (~20K)
- Average price ≈ $811K
- Most properties have 2–3 bedrooms
- Property sales peaked in recent years

## 📊 Dashboard Highlights
- State-wise property distribution
- Price trends over years
- Market size comparison
- Bedroom & bathroom distribution
